# Construction & Demolition Waste Smart Vision System

Goal: Detect, classify, and analyze construction & demolition waste (CDW) materials from images to support automated recycling and smart waste-management decisions.

Main components:
- Object detection on CDW piles (YOLO-based).
- Material classification (MobileNetV2-based).
- Smart analytics: material ratios, confidence statistics, batch-level insights.
- Deployment: Gradio demo UI and optional FastAPI backend.
